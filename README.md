# Aktivitet 4 - Databasanslutning & SQL

## Beskrivning

Detta projekt är utvecklat i PHP och MySQL.

Användaren loggar in med ett enkelt sessionsbaserat
inloggningssystem och kan därefter spara meddelanden
i en MySQL-databas.

## Funktioner

- Inloggning med PHP-sessioner
- Databasanslutning med MySQLi
- Lagring av meddelanden i MySQL
- Hämtning av meddelanden från databasen
- SQL INSERT
- SQL SELECT
- Prepared Statements
- Skyddade sidor för inloggade användare
- Utloggning

## Starta projektet

1. Starta Apache och MySQL i XAMPP.
2. Öppna phpMyAdmin.
3. Skapa databasen aktivitet4_db.
4. Skapa tabellen messages.
5. Besök:

http://localhost/aktivitet4
